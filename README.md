#  YouTubeDownload

YoutubeDownload is an GUI for the Command Line Application "youtube-dl"

## Dependencies
- youtube-dl <br>
Install it with `brew install youtube-dl`
- ffmpeg <br>
Install it with `brew install ffmpeg`
